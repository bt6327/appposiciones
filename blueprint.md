
# Project Overview

This is a Next.js application that provides a "Flow Tracker," a Kanban-style board to help you manage tasks and workflows. Users can create columns to represent different stages of a process and add cards to those columns to represent individual tasks. The application supports drag-and-drop functionality, allowing users to intuitively move tasks from one stage to another.

# Implemented Features

*   **Column Creation:** Users can add new columns to the board to represent different stages of their workflow.
*   **Task Creation:** Users can add new tasks (cards) to any column.
*   **Drag-and-Drop Tasks:** Tasks can be easily moved between columns using a drag-and-drop interface.
*   **Task Deletion:** Tasks can be deleted from the board.

# Design and Styling

*   **Layout:** The application features a horizontal layout, with columns arranged from left to right. The layout is responsive and adapts to different screen sizes.
*   **Styling:** The application uses Tailwind CSS to create a clean and modern design. The color scheme is based on a light and dark theme, with clear visual distinctions between columns and tasks.
*   **User Interface:** The interface is designed to be intuitive and user-friendly, with clear buttons and interactive elements.
